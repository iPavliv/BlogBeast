# BlogBeast

## Description
This is the source code of BlogBeast app, social network.
Share your mind, discuss your interests with other people, get a lot of content.

## Technologies
* Python (3.8.1)
* Flask (1.1.2)
* React (16.13.1)

```
start back.bat & start front.bat
```
